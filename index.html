Cómo Utilizar Git

En este tutorial, te guiaré a través de los pasos básicos para utilizar Git, desde la inicialización de un repositorio hasta la creación de ramas y la conexión a un repositorio remoto en GitHub.

1. Inicialización del Repositorio Local

Lo primero que debemos hacer es inicializar un repositorio local. Abre tu terminal y navega al directorio donde deseas crear tu proyecto. Luego, ejecuta el siguiente comando:

$ git init

Este comando crea un nuevo repositorio Git en tu directorio actual. Puedes verificar que se ha creado correctamente revisando el contenido del directorio:

$ ls -a

Deberías ver una carpeta oculta llamada .git, que contiene toda la configuración de Git para este proyecto.

2. Creación de Ramas (Branches)

Una de las características más poderosas de Git es la capacidad de trabajar con ramas. Las ramas te permiten trabajar en nuevas funcionalidades sin afectar la rama principal (main). Para crear una nueva rama, usa el siguiente comando:

$ git branch nombre-de-la-rama

Para cambiar a esa nueva rama, usa:

$ git checkout nombre-de-la-rama

También puedes crear y cambiar a una nueva rama en un solo paso:

$ git checkout -b nombre-de-la-rama

Una vez que termines de trabajar en tu rama, puedes combinarla con la rama principal (main) utilizando el comando merge:

$ git checkout main
$ git merge nombre-de-la-rama

3. Conectar el Repositorio Local con un Repositorio Remoto

Ahora que ya tienes un repositorio local, puedes conectarlo a un repositorio remoto en GitHub para colaborar con otras personas o simplemente para hacer un respaldo. Para ello, primero crea un repositorio en GitHub, luego conecta tu repositorio local al remoto con los siguientes comandos:

$ git remote add origin https://github.com/tu-usuario/tu-repositorio.git
$ git push -u origin main

Ahora, cada vez que realices cambios en tu proyecto, puedes subirlos a GitHub usando:

$ git push
